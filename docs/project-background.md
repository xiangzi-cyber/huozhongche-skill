# 项目背景

## 当前项目名称

项目当前统一名称为：`AI火种车数智课堂`

此前在部分资料、推文与说明文档中使用过 `AI火种车` 或相关表述，当前建议在公开说明、项目介绍、协作文档中逐步统一为最新名称。

## 项目定位

`AI火种车数智课堂` 是一个围绕 AI 教育普及、在地化教学与公益协作展开的项目。

根据现有项目背景资料，可以概括为：

- 由 WaytoAGI 教育版发起
- 通过 AI 志愿老师推动 AI 教育向偏远地区普及
- 关注城乡数字鸿沟的缩小与教育资源的连接
- 强调深入具体学校、具体场域开展“在地化”教学

## 为什么这个背景对 Skill 很重要

这个 Skill 不是脱离项目背景的通用写作工具，它的很多规则都与项目本身的叙事气质密切相关，例如：

- 为什么强调“平视视角”
- 为什么要避免怜悯叙事和扶贫标签
- 为什么配图要同时考虑隐私保护与现场感
- 为什么系列稿件既要统一风格，又要避免表达同质化

换句话说，这个 Skill 的写作要求，本质上是在服务 `AI火种车数智课堂` 的项目表达。

## 在仓库中的作用

本说明文档的作用是帮助协作者快速理解：

- 这个项目现在应该如何命名
- 这个 Skill 的价值为什么不仅是“写稿”
- 为什么仓库里会同时出现写作流程、讲解页面和案例目录
