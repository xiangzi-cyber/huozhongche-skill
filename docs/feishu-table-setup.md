# 飞书多维表格搭建指南

> 将素材表格模板导入飞书多维表格的操作指南。

---

## 导入步骤

1. 在飞书文档中选择「新建多维表格」
2. 点击「从 Excel 导入」
3. 选择 `火种车素材整理系统_通用版v3.xlsx`
4. 分别导入以下三个 sheet 为三个数据表：
   - 素材明细表
   - 写稿亮点汇总
   - 活动总览

注意：「使用说明」和「素材整理助手提示词」两个 sheet 不需要导入，它们是给人看的文档。

---

## 导入后设置

### 素材明细表

| 字段名 | 飞书字段类型 |
|-------|-----------|
| 活动编号 | 文本 |
| 学校名称 | 文本 |
| 活动日期 | 日期 |
| 环节 | 单选（前期准备/旅途出发/入校/学校背景/教师AI讲座/教师实操工作坊/中午展示/学生AI讲座/学生实操工作坊/收尾/其他） |
| 时间段 | 文本 |
| 模块 | 文本 |
| 素材点 | 文本 |
| 具体内容 | 多行文本 |
| 写稿价值 | 多行文本 |
| 素材类型 | 单选（文字/图片/文字+图片/视频/语音/其他） |
| 配图描述 | 多行文本 |
| 素材状态 | 单选（待整理/已整理/已用于写稿/存疑待确认） |
| 备注 | 多行文本 |

### 写稿亮点汇总

| 字段名 | 飞书字段类型 |
|-------|-----------|
| 活动编号 | 文本 |
| 学校名称 | 文本 |
| 类型 | 单选（事件链/最强素材点/主轴/可写句子） |
| 内容 | 多行文本 |
| 来源环节 | 文本 |
| 优先级 | 单选（高/中/低） |
| 是否已用 | 单选（是/否） |
| 备注 | 多行文本 |

### 活动总览

| 字段名 | 飞书字段类型 |
|-------|-----------|
| 活动编号 | 文本 |
| 学校名称 | 文本 |
| 地理位置 | 文本 |
| 活动日期 | 日期 |
| 学校特点 | 多行文本 |
| 素材状态 | 单选（待整理/整理中/已完成） |
| 成稿状态 | 单选（未开始/写稿中/已发布） |
| 发布链接 | 链接 |
| 备注 | 多行文本 |

## 推荐视图

| 视图名称 | 视图类型 | 分组/筛选 | 用途 |
|---------|---------|---------|------|
| 按环节看素材 | 看板视图 | 按「环节」分组 | 浏览每个环节有哪些素材 |
| 按状态看进度 | 看板视图 | 按「素材状态」分组 | 追踪整理进度 |
| 写稿素材池 | 表格视图 | 筛选「写稿价值」非空 | 写稿时快速找可用素材 |
| 高优先级素材 | 表格视图 | 筛选「优先级=高」 | 优先使用的素材 |
