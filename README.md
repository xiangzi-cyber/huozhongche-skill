# huozhongche-skill

一个用于“AI火种车”系列文章与推文写作的可复用 Codex Skill。

## 仓库内容

- `SKILL.md`
  - Skill 主说明文件，定义完整写作流程与规则。
- `scripts/dedup_check.py`
  - 用于表达去重分析的辅助脚本。
- `references/`
  - 预留参考资料目录。
- `templates/`
  - 预留模板目录。
- `docs/skill-understanding.html`
  - Skill 结构理解说明页面。
- `docs/editor-training-deck.html`
  - 面向协作老师的系统讲解版页面。
- `docs/editor-training-deck.md`
  - 对应讲稿版说明。

## 这个 Skill 解决什么问题

它不是单纯教“怎么把活动写出来”，而是一套从素材梳理到终稿交付的写作系统，重点包括：

- 信息梳理与缺口识别
- 故事主线提炼
- 结构与标题风格选择
- 初稿深化与平视视角
- 配图策略与隐私保护
- 表达去重、错别字检查与终稿交付

## 核心特点

- 把写稿过程拆成清晰的 6 个步骤，减少协作时的混乱感。
- 强调“平视视角”，避免怜悯叙事和扶贫标签化表达。
- 把配图建议纳入写作流程，而不是发稿前临时补图。
- 通过表达去重机制，避免系列稿件语言风格越来越像。

## 如何使用

把本仓库作为本地 skill 安装到 Codex 技能目录后，在与“AI火种车”相关的写作任务中调用即可。

如果你已经在使用本地 Codex skill 目录，通常结构类似：

```text
$CODEX_HOME/skills/huozhongche-skill
```

## 附加文档

为了方便协作和培训，仓库里附带了两类说明文档：

- Skill 理解说明
  - 用来帮助协作者快速理解这套写作系统的结构与重点。
- 讲解页面
  - 用来向同级老师解释这套系统为什么能减轻焦虑、如何按点检查推进。
