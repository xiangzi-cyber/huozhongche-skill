# 第一批 PDF 案例稿

这里收录的是第一批已经处理完成的 PDF 案例稿。

这些文件适合用于：

- 作为公开示例稿展示
- 给新协作者理解系列文章风格
- 对照 Skill 的写作方法做拆解
- 作为后续表达去重和文风比较的参考样本

## 当前收录案例

1. `第一站：把未来带进教室，让AI成为课堂“第四要素”.pdf`
   - 适合看点：作为系列起笔，适合观察“项目开场时如何快速建立主题、场景和叙事基调”。

2. `彩云之南第二站：在太平新城，把 AI 变成课堂里的“新常客”_20260302194053_副本.pdf`
   - 适合看点：适合观察“如何把技术引入课堂写得自然”，以及如何把“新常客”这种意象写成文章的统一抓手。

3. `彩云之南第三站：当 AI 走进“1133+n”的校园——姚安县太平中心小学_20260302194056_副本.pdf`
   - 适合看点：适合观察“如何处理学校已有教学体系与 AI 进入课堂之间的关系”，也适合参考“概念型信息如何落地成叙述”。

4. `火种车第四站：在官屯，AI让梦想变得“天花乱坠”_20260302194103_副本.pdf`
   - 适合看点：适合观察“如何把原本可能带有负面色彩的词语做正向翻转”，这篇很适合作为步骤四里“词语翻转解读”的示例。

5. `AI火种车第五站：边境线上的AI魔法——怒江大峡谷的跨越与守望_20260302194108_副本.pdf`
   - 适合看点：适合观察“地域感、边境感与项目主题如何同时成立”，也适合参考更有场域张力的标题和开头写法。

6. `AI火种车收官站：一所“编外”中学，一位“推墙”校长，和一场关于未来的共同建构_20260302194115_副本.pdf`
   - 适合看点：适合观察“人物、学校气质与项目价值如何在收官稿里汇合”，也适合拿来参考如何写得更像阶段总结而不失故事性。

## 如何挑着看

如果你是第一次接触这个 Skill，可以按下面顺序看：

- 想先理解系列起笔方式：先看第一站
- 想看课堂场景怎么写得自然：看第二站
- 想看概念和学校体系如何写进正文：看第三站
- 想看词语翻转和情绪转折：看第四站
- 想看地域场域感更强的写法：看第五站
- 想看收官稿如何兼顾人物、结构和总结意味：看收官站
