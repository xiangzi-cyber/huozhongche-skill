# 案例目录说明

这个目录用于存放 `AI火种车数智课堂` 相关的写稿案例。

建议优先放：

- 已发布稿件的精选样本
- 有代表性的初稿 / 改稿对照
- 不同学校、不同场域、不同叙事结构的示例稿
- 用于内部培训的拆解案例

## 推荐分类方式

你可以按下面这些方式组织案例：

- 按学校或站点分类
- 按时间或期次分类
- 按稿件类型分类
  - 正式推文
  - 志愿者集结稿
  - 阶段总结稿
  - 训练案例

## 推荐文件形式

- `.md`
- `.pdf`
- 体量适中的 `.docx`

## 不建议直接上传的内容

- 单个超大压缩包
- 未整理的原始素材全集
- 体量过大的图片或扫描件合集

如果案例包很大，建议：

- 先拆成精选案例
- 或在仓库中放一个下载说明 / 云盘链接说明
