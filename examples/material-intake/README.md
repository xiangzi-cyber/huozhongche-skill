# 前置素材收集表案例

这个目录用于存放写作系统进入正式成稿之前的“前置素材收集表”案例。

这类材料的作用不是直接展示成稿，而是帮助协作者先把“写稿需要的关键输入”收集完整。

## 这一部分为什么重要

在这套写作系统里，很多后续问题其实都来自前置素材不完整，例如：

- 活动日期缺失
- 课表缺失
- 关键人物姓名和身份不清楚
- 缺少老师和学生的原话
- 只有零散印象，没有可组合成故事的“积木”

所以，前置素材收集表可以理解为：

> 它不是附属工具，而是整个写作系统的起点。

## 当前收录内容

- `火种车推文写作素材表-昆明常家庄小学.xlsx`
  - 一个实际使用过的素材表样例，可用于理解前置素材大概需要收集到什么程度。

- `飞书多维表格搭建傻瓜教程 (V8 - 成稿反推版).md`
  - 一份从成稿反推素材字段设计的搭建说明，适合用来理解飞书多维表格版本为什么这样设计。

## 适合怎么使用

- 给新协作者解释“写稿前到底该先收什么”
- 给团队统一素材收集标准
- 作为飞书表单 / 多维表格字段设计的参考模板
- 帮助理解“为什么后续能写出故事，前面必须先收对信息”

## 和 Skill 的关系

如果说 `SKILL.md` 解决的是“怎么写”，那么这一目录里的材料解决的是：

> “写之前，应该先把什么收齐。”
